# Project AirBnB 🏠

A full-stack web application inspired by Airbnb, built using the **MVC (Model-View-Controller)** design pattern to ensure a clean, modular, and scalable codebase.

## 🚀 Features
- **User Authentication**: Secure signup and login functionality.
- **Property Listings**: Browse, view, and search for various accommodations.
- **MVC Architecture**: Clear separation of concerns between data, logic, and UI.
- **Responsive Design**: Fully functional on desktop, tablet, and mobile devices.

## 🛠️ Tech Stack
- **Frontend**: EJS (Embedded JavaScript templates), HTML5, CSS3,bootstrap, JavaScript
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (with Mongoose ODM)
- **Architecture**: MVC (Model-View-Controller)
